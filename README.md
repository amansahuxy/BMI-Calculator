# ⚖️ BMI Calculator

A simple BMI (Body Mass Index) Calculator built using HTML and JavaScript. Users can enter their weight and height to calculate their BMI and view their BMI category instantly.

## 🚀 Features

* Calculate BMI from weight and height
* Display BMI value
* Show BMI category
* Simple and beginner-friendly interface

## 🛠️ Technologies Used

* HTML5
* JavaScript

```text
BMI-Calculator/

└── README.md
```

## ▶️ How to Run

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## 🧠 How It Works

1. Enter your weight in **kilograms (kg)**.
2. Enter your height in **centimeters (cm)**.
3. Click the **Calculate** button.
4. The app calculates your BMI using the formula:

```text
BMI = Weight (kg) / Height² (m²)
```

5. The BMI value and category are displayed.

## 📊 BMI Categories

| BMI Range    | Category       |
| ------------ | -------------- |
| Below 18.5   | Underweight    |
| 18.5 – 24.9  | Healthy Weight |
| 25 – 29.9    | Overweight     |
| 30 and above | Obesity        |
